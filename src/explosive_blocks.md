# Explosive Blocks

```lua
{ 17000
    features=EXPLODE

    explodeDamage=50
    explodeRadius=50
    explodeStdDev=10  -- *
    explodeFaction=98 -- The specified faction will not take damage from the explosion.
}
```

*For the explodeStdDev field, see [Standard Deviation Fields](./standard_deviation_fields.md).
