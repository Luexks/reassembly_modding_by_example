# Explosive Blocks

```lua
{17000
    features=EXPLODE

    explodeDamage=50
    explodeRadius=50
    explodeStdDev=10 -- *
    explodeFaction=98 -- Makes the specified faction not take damage from the explosion.
}
```

*For the explodeStdDev field, see [Standard Deviation Fields](./standard_deviation_fields.md).
