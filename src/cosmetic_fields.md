# Cosmetic Fields
All of the following features are purely decorative and do nothing.
## Internal Lines
Block always renders lines.
```lua
{17000
    features=INTLINES
}
```
## No Icon
Block doesn't renders icons.
```lua
{17000
    features=NOICON
}
```
## No Recolor
Block is not recolored by color schemes.
```lua
{17000
    features=NORECOLOR
}
```
